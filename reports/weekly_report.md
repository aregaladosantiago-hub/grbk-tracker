## How to read it

- **Active listings** = current supply visible on tracked brand sites.
- **New since prior** = homes visible now that were not visible in the prior comparable snapshot.
- **Removed / Sold Proxy** = homes visible in the prior comparable snapshot that are no longer visible; useful for sales traction, but not confirmed sold.
- **Added this week** = unique homes that appeared during the current fixed weekly period.
- **Net Change** = added minus removed/sold-proxy homes.
- **Daily movement this week** = total added/removed movement for each successful snapshot inside the current weekly period.
- **Weekly history log** = frozen weekly periods from the first saved snapshot, so Week 1 and Week 2 remain auditable instead of being overwritten.

## Notes

- New/removed counts only compare brands present in both snapshots, so adding a new tracked builder does not count its entire inventory as newly listed homes.
- Newly configured brands show `no_rows` until their first successful snapshot is captured.
- A removed listing can mean sale, relisting, URL change, or a temporary website change, so use it as a directional sales-traction signal.
- If a scheduled scrape is missed, the next daily movement row covers the change since the prior successful snapshot.
- Price-cut metrics were removed from the headline report because they measure builder discounting, not weekly sales traction.
